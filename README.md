This is the package of math games. User interracts to games via cli to install and run games.  
Every game has personal command to run:  
```brain-prime ```  
```brain-even ```  
```brain-gcd ```  
```brain-calc ```  
```brain-progression ```  
Every game has tasks and check your answers  


To install games run ```make package-install ```
